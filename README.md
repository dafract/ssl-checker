# ssl-cheker

## How to use
run the following command
```sh
$ bash ssl-check.sh www.example.com
```

then you will get the following response
```
93.184.216.34

notAfter=Feb 13 23:59:59 2024 GMT
    Verify return code: 0 (ok)
remaining: 270 days
```
